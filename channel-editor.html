Welcome to the CHANNEL EDITOR

The channel editor is a web-based application where you can create, edit, and update the content that appears in your channel. Any changes you make in the channel editor will become available to your visitors when you click the save button.

When you’re first starting out, however, you may not want everyone to see what you’re working on. For that reason, your channel is set to “test” mode when you first launch the channel editor. In test mode, only you will be able to see your channel and its contents on the website and in the MyTourGuide mobile app. When you are ready to show it off to the world, you can switch it to “public” mode in the settings tab.

Indoor Maps

If you have indoor locations that you want to make accessible to visitors, you can upload your own custom maps and use them in your channel, itinerary, or tours. If you don’t have any need to feature indoor maps, you can skip this section. Your custom map should be saved as a jpg with a minimum dimension of 1024x1024 pixels and a maximum file size of 5 MB. To add a map, first place a building on the outside map. Then add your map(s) to the building. By default, new buidlings and maps are set to the “inactive” state, which means they are hidden from view to your visitors. To make them visible, change the building or map to “active”. If you add more than one map to a building, you can make one of the maps the primary map. The prmary map is the default map that the visitor will see when viewing maps in that building.

What is an API?

The MyTourGuide API is like a key that opens a locked door on the MyTourGuide website where you can upload your existing POI data. So, If you already have a database of information about attractions, restaurants, etc., the API allows you to upload that information with the click of a button.

To use the MyTourGuide API, you will need to request an API key. Once you have your API key, the person who manages your website or POI database will be able to use the key to establish a connection between your database and MyTourGuide, and then upload the POI data to your channel.

Here is a summary of what you need to do to get started:

  1. Start by requesting an API key
  2. Give your web developer or POI database manager access to your channel account and permission to upload your POI data using the MyTourGuide API

POIs

Do you already have a database of points of interest (POIs)? If so, then you can upload them to your channel with the click of a button. Learn more.

Don’t have a database of POIs? That’s not a problem. Create as many POIs as you like right here in the channel editor. New POIs that you create are "inactive" or hidden to visitors by default. Once the required information (highlighted in red) is added for the POI, you can switch its status to “active” and visitors will be able to see it in your channel. If you ever need to hide a POI from visitors, just change the POI status to inactive.
